# CSS样式统一

#### 隐藏滚动条
```css
::-webkit-scrollbar {
    display: none;
}
```

#### 文本溢出显示省略号
```less
.text-overflow-1 {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

.text-overflow-2 {
  overflow: hidden;
  text-overflow: ellipsis;
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
}
```
